Custom Field Rules

Rules are checked in order — first match wins. Use these to target specific fields by their attributes. Set action to "Skip" to leave a field empty.

Website Presets

Create presets for specific websites. When the URL matches, preset rules are applied before custom rules. Use * as wildcard in URL patterns.
